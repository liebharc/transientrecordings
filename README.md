A very simple recorder for musicians: Take a recording, listen to yourself, learn and repeat. And if you have a good take you can share it with yourself or friends to persist it.

The main reason this exists is that finding an ad-free & tracking-free app is hard these days.

Hosted at: https://transientrecordings.vercel.app/

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
